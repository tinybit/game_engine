About This Guide

The SOFTIMAGE®|XSI® FTK Developer’s Guide provides information about the dotXSI file format and the FTK (File Toolkit).

  • Chapter 2: Introducing the dotXSI File Format—Overview of the dotXSI file format, what you can do, and where to look to find out how.
  • Chapter 3: Getting Started—What you need to know about your project and compilation settings before you start coding.
  • Chapter 4: The dotXSI Core API—Provides information on the fundamental tools and classes available with the dotXSI file format.
  • Chapter 5: The FTK API—Provides information on the tools and classes available with the FTK API. Includes many examples.
  • Chapter 6: Template Reference—Provides details about the data stored in each of the dotXSI file templates.
  • Chapter 7: Version Information—Relates version-specific information from version 1.3 to the current template.
  • Chapter 8: Template Archives—Template reference for obsolete templates.
  • Chapter 9: Case Studies—Examples of how you can use the dotXSI file format.
