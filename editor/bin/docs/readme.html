------------------------------- Shortcuts -------------------------------
  • Общие
    • Открыть файл: 
      		•  Ctrl-O
    • Сохранить файл: 
      		•  Ctrl-S
    • Показать диалог настроек редактора: 
      		•  Ctrl-Enter
    • Показать файл справки: 
      		•  F1
    • Режим отображения "wireframe": 
      		•  F2
    • Режим отображения "solid": 
      		•  F3
    • Режим отображения "solid & wireframe": 
      		•  F4
    • Отсечение невидимых поверхностей (on/off): 
      		•  F5
    • Показать "bounding box" (on/off): 
      		•  O
    • Показать "bounding sphere" (on/off): 
      		•  P
    • Запустить игровой движок: 
      		•  F9
    • Закрыть программу: 
      		•  Alt-F4

  • Файлы EFN (animation)
    • Выделение кадров анимации: 
      		•  удерживая "Ctrl" выделить кадры бегунком, отпустить Ctrl
    • Редактирование "global radius": 
      		•  выделить в дереве объектов узел Info->GlobalRadius, удерживая "С" и правую
      кнопку мыши, задать значение. Отпустить правую кнопку мыши
    • Выделить все кадры: 
      		•  Alt-A
    • Снять выделение со всех кадров: 
      		•  Alt-Z
    • Удалить выделенные кадры: 
      		•  Shift-Del

  • Файлы EFN (meshlist)
    • Скрыть выделенный объект: 
      		•  H
    • Показать все объекты: 
      		•  Shift-H
    • Скрыть все кроме выделенного объекта: 
      		•  Ctrl-H
    • Перейти в режим редактирования вершин выделенного объекта: 
      		•  V
    • Выйти из режима редактирования вершин выделенного объекта: 
      		•  V
    • Привязка к оси X: 
      		•  1
    • Привязка к оси Y: 
      		•  2
    • Привязка к оси Z: 
      		•  3
    • Привязка к осям XY: 
      		•  4
    • Привязка к осям XZ: 
      		•  5
    • Привязка к осям YZ: 
      		•  6
    • Добавить вершину к уже выделенным: 
      		•  удерживать "Ctrl", выделить вершины
    • Удалить вершину из уже выделенных: 
      		•  удерживать "Alt", выделить вершины
    • Сбросить выделение вершин: 
      		•  выделить пустое место
    • Переместить выделенные вершины (согласно привязке по осям): 
      		•  удерживать "W", удерживать правую кнопку
      мыши, передвинуть вершины, отпустить правую кнопку мыши
    • Редактирование поворота нулика по оси Y: 
      		•  удерживать "Q", удерживать правую кнопку
      мыши, повернуть нулик, отпустить правую кнопку мыши
    • Редактирование позиции нулика по осям XZ: 
      		•  удерживать "W", удерживать правую кнопку
      мыши, переместить нулик, отпустить правую кнопку мыши
    • Редактирование позиции нулика по оси Y: 
      		•  удерживать "E", удерживать правую кнопку
      мыши, переместить нулик, отпустить правую кнопку мыши
